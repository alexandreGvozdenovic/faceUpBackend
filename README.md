# Face Up (backend)
Application mobile qui grâce à l'IA de reconnaissance de Visage de Microsoft Azure donne des informations relatives à un visage
## Technologies utilisées
- Node.js (Express framework)
- Microsoft Azure Faces (API)
- Cloudinary
